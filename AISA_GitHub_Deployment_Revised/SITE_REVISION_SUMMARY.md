# AISA Website Messaging Revision

## Strategic change
AISA now leads as a senior industrial systems advisory company serving manufacturing, water, wastewater, utilities, and OEM organizations. Diagnostic advisory and FaultAssist™ remain important capabilities, but no longer define the limits of the company.

## Major revisions
- Rebuilt the homepage around problems, projects, decisions, commissioning, modernization, and workforce capability.
- Added a full Advisory Services page with six service categories.
- Expanded all audience language to include water, wastewater, utilities, and OEMs.
- Repositioned FaultAssist™ as an optional proprietary capability rather than the assumed outcome of every engagement.
- Broadened contact forms and calls to action from recurring faults to any industrial problem, project, or decision.
- Updated navigation, metadata, footer language, and sitemap.
- Preserved the existing visual system, assets, Formspree endpoint, and static GitHub-compatible structure.

## Deployment
Replace the existing repository site files with the contents of this folder, commit, and push through the current GitHub deployment workflow.
